Syncthing is a file synchronization tool that allows you to keep files and folders in sync across multiple devices. It works by creating a shared folder on one device and adding other devices to the folder. Any changes made to the files in the folder on one device are automatically synced to all the other devices in the folder. Syncthing is free, open-source, and available for Windows, Linux, macOS, Android, and other platforms.

One of the key advantages of Syncthing is its security. It uses encryption to protect communication between devices, and all data remains on your own devices, rather than being stored in the cloud. Syncthing also supports versioning, so you can revert to a previous version of a file if needed, and has a user-friendly web interface that makes it easy to manage synchronization between devices.

To use Syncthing you need to have basic knowleage in windows.
